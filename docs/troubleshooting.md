# Решение проблем

## Приложение не запускается на Windows

- Убедитесь, что скачана актуальная сборка из GitHub Releases.
- Проверьте, не заблокировал ли файл SmartScreen, Microsoft Defender или другой антивирус.
- Сверьте SHA-256 с `checksums.txt` и проверьте ссылку VirusTotal из `SECURITY.md`.
- Попробуйте запустить приложение от имени пользователя с правами доступа к устройству.

## AppImage не запускается на Linux

Проверьте права на запуск:

```bash
chmod +x MIK32_Uploader-x86_64_v1.0.0.AppImage
```

Если система сообщает об ошибке FUSE, установите пакет совместимости AppImage для вашего дистрибутива.

## OpenOCD не найден

Укажите полный путь к `openocd.exe` или добавьте OpenOCD в `PATH`.

## Не найден cfg

Выберите правильный каталог `scripts` OpenOCD или конкретный cfg-файл.

## Устройство не определяется

- Проверьте USB-кабель.
- Подключите устройство напрямую к компьютеру без USB-хаба.
- Убедитесь, что устройство получает питание.
- Проверьте выбранный `interface cfg`.
- Проверьте права доступа к USB-устройству на Linux.

## Связь нестабильна

- Уменьшите `Speed (kHz)`, например до `100` или `200 kHz`.
- Проверьте питание платы.
- Проверьте кабель.
- Проверьте выбранный `interface cfg`.

## Прошивка не загружается

- Проверьте формат файла прошивки: `HEX`, `BIN` или `ELF`.
- Убедитесь, что выбран правильный `MCU Type`.
- Проверьте `Boot Mode`.
- Повторите операцию после переподключения устройства.
