# Релизы и версии

Бинарные сборки MIK32 Flasher публикуются как assets в GitHub Releases. В репозитории хранятся документация, контрольные суммы, сведения о проверке и автотесты.

Рекомендуемая схема версий:

```text
vMAJOR.MINOR.PATCH
```

Примеры:

```text
v1.0.0
v1.1.0
v2.0.0
```

Текущие имена файлов:

```text
mik32_upload_v1.0.0.exe
MIK32_Uploader-x86_64_v1.0.0.AppImage
checksums.txt
```

## Что обновлять при новом релизе

- загрузить новые бинарники в GitHub Release;
- обновить `checksums.txt`;
- обновить `SECURITY.md` с SHA-256, статусом сборок и ссылками на проверки;
- обновить `CHANGELOG.md`;
- обновить ссылки в `README.md` и `docs/installation.md`, если изменились имена файлов;
- проверить запуск Windows-сборки и Linux AppImage.
- при наличии тестовой платы прогнать автотесты из `tests/`.
