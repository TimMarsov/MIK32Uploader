# Changelog

Все заметные изменения MIK32 Flasher будут документироваться в этом файле.

## [1.0.1] - 2026-07-10

### Added

- Добавлен номер версии `v1.0.1` в верхнюю панель графического интерфейса.
- Добавлены CLI-аргументы `-v` и `--version` для вывода версии приложения.
- Добавлено определение имени OpenOCD target через команды `target names` и `targets` при маскировании CPU interrupt-регистров.

### Changed

- Версия приложения обновлена до `1.0.1` для Windows и Linux-сборок.
- Запись регистров `mstatus` и `mie` теперь выполняется через найденный OpenOCD target с fallback на `mik32.cpu` и `riscv.cpu`, что улучшает совместимость с разными target-конфигами.
- Сообщения OpenOCD о JTAG scan chain теперь отображаются как предупреждения, а строки `tap/device found` не подсвечиваются как ошибки из-за слова `invalid`.

## [1.0.0] - 2026-07-04

### Added

- Первая публичная бинарная сборка для Windows.
- Первая публичная бинарная сборка для Linux в формате AppImage.
- Добавлены контрольные суммы SHA-256 и сведения для проверки релизных файлов.
- Добавлены публичные автотесты записи и верификации прошивок для RAM, EEPROM и SPIFI Flash.
