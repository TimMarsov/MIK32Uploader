# Безопасность и проверка файлов

MIK32 Flasher распространяется в виде публичных бинарных сборок. Исходный код проекта закрыт и не публикуется в этом репозитории, поэтому проверка релизов вынесена отдельно: для каждого публичного бинарного файла должна быть опубликована контрольная сумма SHA-256 и, когда возможно, ссылка на внешнюю проверку антивирусами.

## Проверка SHA-256

Windows PowerShell:

```powershell
Get-FileHash -Algorithm SHA256 .\mik32_upload_v1.0.0.exe
Get-FileHash -Algorithm SHA256 .\MIK32_Uploader-x86_64_v1.0.0.AppImage
```

Linux:

```bash
sha256sum -c checksums.txt
```

## Windows SmartScreen и Defender

Windows SmartScreen или Microsoft Defender могут предупреждать о новых неподписанных бинарных файлах, особенно пока у приложения нет накопленной репутации. К любому предупреждению стоит относиться внимательно, но перед решением проверьте сам файл:

1. Скачивайте бинарные файлы только из этого репозитория или из официального GitHub release.
2. Сравните SHA-256 с `checksums.txt` и таблицей выше.
3. Проверьте ссылку VirusTotal, если она опубликована.
4. Если хэш не совпадает, не запускайте файл.

## Сообщить о проблеме безопасности

Если вы видите подозрительное обнаружение, несовпадение контрольной суммы или неожиданное поведение, создайте issue и укажите:

* точное имя файла и версию;
* вашу операционную систему;
* SHA-256, который получился у вас;
* детали обнаружения антивирусом или VirusTotal;
* шаги для воспроизведения проблемы.
